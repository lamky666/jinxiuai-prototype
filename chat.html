用户需要 ① 兼容 OpenAI SDK ② 最小可运行示例 ③ Python 代码。我应该用 base_url 指向锦绣的中转地址，api_key 用锦绣生成的 key，并给出安装 pip install openai 的前置步骤。
没问题，先安装官方 SDK： 
# 1. 安装
pip install openai

# 2. 最小示例
from openai import OpenAI

client = OpenAI(
    base_url="https://api.jinxiu.ai/v1",  # 锦绣中转
    api_key="sk-jinxiu-你的key",
)

resp = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "你好"}],
)
print(resp.choices[0].message.content)
关键点：
  • base_url 指向锦绣的网关，其余代码 与 OpenAI 官方 SDK 100% 兼容，无需改业务逻辑；
  • API Key 在 控制台 → API Key 创建；
  • 模型名 gpt-4o / claude-3.5-sonnet 可混用，详见 API 文档。
