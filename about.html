关于我们

锦绣AI 是一个面向中文开发者的 AI 算力中转平台，致力于让每一次 API 调用都透明、可追溯、高性价比。

我们的定位

我们不做「黑箱」中转站。通过模型透明溯源、按量实时计费、对话内容不落库三项核心机制，为个人开发者、初创团队和企业提供可信赖的 AI 算力服务。

核心原则

1

透明

每次调用返回实际模型、Token 用量和费用，无隐藏扣费。

2

可信

不预收大额套餐，余额随时可查，对话内容不存储。

3

高性价比

对接 DeepSeek、智谱、通义、OpenAI、Anthropic 等渠道，自动路由到最优方案。

合规与隐私

我们正在办理 ICP 备案与算法备案。平台仅记录 API 调用所需的 Token 用量和费用，不存储用户的对话内容、prompt 或代码。
